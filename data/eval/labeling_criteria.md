# W07 평가셋 라벨링 기준

> 명세서 §13 "판정 기준의 주관성 → 정확도 측정 신뢰도 저하" 대응 문서.
> 대상: `data/eval/w07_eval.jsonl` (합성 광고/상담 스크립트 30건)
>
> **상태: 라벨 확정 (2026-07-13)** — §4의 쟁점 4건을 조문 원문과 대조해 확정했다.

---

## 1. 이 문서가 필요한 이유

"이 문구가 위반인가"는 사람마다 다르게 답할 수 있다. 기준을 적어두지 않으면 판정
정확도 수치는 "모델이 나와 얼마나 비슷하게 생각하는가"를 잴 뿐, 재현도 검증도 안 된다.
그래서 **라벨을 붙인 규칙을 먼저 고정**하고, 그 규칙으로 30건을 라벨링했다.

---

## 2. 핵심 규칙

### 규칙 1 — 위반은 "조문이 열거한 행위"에 해당할 때만

금소법 6대 판매규제는 각 조문이 금지·의무 행위를 **열거**하는 구조다. 문구가 나쁘다는
인상이 아니라, **그 조문이 열거한 행위에 해당하는지**로 라벨을 정한다.

| 조문 | 열거된 행위 |
|---|---|
| 제17조 적합성 | 소비자 정보 미파악 권유 / 파악된 정보에 부적합한 상품 권유 / 확인 내용 미제공 |
| 제18조 적정성 | (권유 없는 청약에서) 정보 미파악 / 부적정 사실 미고지·미확인 |
| 제19조 설명의무 | 중요사항 설명 생략 / 설명서 미교부·이해 미확인 / 거짓·왜곡 설명 |
| 제20조 불공정영업 | 끼워팔기 강요 / 부당 담보·보증 요구 / 편익 요구 / 부당 중도상환수수료 |
| 제21조 부당권유 | 단정적 판단 제공 / 거짓 고지 / 근거 없는 비교 / 거부 후 재권유 / 불초청 권유 |
| 제22조 광고 | 광고 필수 포함사항 누락 / 광고에서의 오인 유발 표현 |

### 규칙 2 — 하나의 행위를 여러 원칙에 돌려쓰지 않는다

한 문구에 문제 행위가 하나 있으면, 그 행위를 열거한 **조문 하나에만** VIOLATION을 붙인다.
예: "대출받으려면 보험도 가입하라"(꺾기)는 제20조의 열거 행위다. 이 사실을 근거로
제17·19·21조까지 VIOLATION을 붙이지 않는다.

**복수 라벨은 (a) 문구에 서로 다른 행위가 실제로 여러 개 있거나, (b) 하나의 행위를
둘 이상의 조문이 함께 열거하고 있을 때만** 붙인다. (b)의 유일한 사례가 아래 규칙 2-1이다.

### 규칙 2-1 — 단정적 표현: 국면 조건부 복수 라벨 (제21조 + 제19조③)

제19조제3항은 "왜곡"을 괄호로 **"불확실한 사항에 대하여 단정적 판단을 제공하거나
확실하다고 오인하게 할 소지가 있는 내용을 알리는 행위"**라고 정의한다. 이는 제21조제1호의
문언과 **동일하다**. 즉 하나의 행위(단정적 표현)를 두 조문이 함께 열거하고 있다.

다만 제19조제3항은 **"제1항에 따른 설명을 할 때"**로 적용이 한정되고, 제19조제1항의 국면은
**"계약 체결을 권유하는 경우 및 소비자가 설명을 요청하는 경우"**다. 그래서 국면으로 갈린다.

| 문구 국면 | 단정적 표현의 라벨 |
|---|---|
| **상담·권유** (= 설명 국면) | 제21조제1호 **+ 제19조제3항** → 부당권유·설명의무 **둘 다 VIOLATION** |
| **광고 카피** | 제19조는 적용 대상이 아님(제1항 국면이 아니다) → 제22조 / 제21조만 |

**국면 판단 기준** — 아래로 상담/광고를 가른다. (자동 분류기 결과는 참고만 하고 문구로 판단)

- **상담·권유**: 특정 고객을 상대로 한 1:1 대화. "고객님", 청약·서명·확인 절차 언급,
  대화체 종결어미, 상대의 응답을 전제한 문장.
- **광고**: 불특정 다수 대상 카피. 상품명·판매사 표기, 캐치프레이즈, 심의필 문구,
  대화 상대가 특정되지 않음.
- 애매하면 **상담으로 본다** — 설명의무를 함께 보게 되어 더 보수적(누락 위험이 작다).

적용 문항: w07-021, w07-022 (둘 다 상담 + 단정적 표현 → 설명의무도 VIOLATION).

### 규칙 3 — 침묵은 위반이 아니다

이 도구가 점검하는 것은 **문구**이지 판매 절차 전체가 아니다. 문구만 봐서는 판매자가
실제로 정보를 파악했는지, 설명서를 줬는지 알 수 없다.

- 문구가 절차를 **생략·회피하겠다고 말하면** → VIOLATION
- 문구에 그 절차 얘기가 **그냥 없으면** → OK (위반의 근거가 아니다)

### 규칙 4 — 적용 국면이 아니면 OK

각 원칙은 적용 국면이 정해져 있다. 국면이 아니면, 문구에 다른 문제가 있어도 그 원칙은 OK다.

- 제17조 적합성 → **권유** 국면
- 제18조 적정성 → **권유 없는 자발적 청약** 국면
- 제19조 설명의무 → **권유·설명** 국면 (광고는 제외)
- 제21조 부당권유 → **권유** 국면
- 제22조 광고 → **광고** 국면 (상담 대화는 제외)

### 규칙 5 — 근거 조항은 규제 체인 전체 (ADR-004)

`evidence_articles`는 같은 규제 사항의 **본법·시행령·감독규정을 모두** 기재한다.
실무에서 근거로 드는 조항은 세 층위 어느 것이든 될 수 있기 때문이다.
(예: 꺾기의 구체적 판단 기준은 본법 제20조가 아니라 감독규정 제14조에 있다.)

> 참고: `evidence_articles`는 W08 인용 검증용 참조 라벨이다. **W07 정확도 측정에는
> 쓰이지 않는다** — W07이 재는 것은 원칙별 verdict 일치율뿐이다.

## 3. 평가셋 구성

| 원칙 | 위반 | 정상(대조) |
|---|---|---|
| 적합성 원칙 | w07-001, 002, 003 | w07-004, 005 |
| 적정성 원칙 | w07-006, 007, 008 | w07-009, 010 |
| 설명의무 | w07-011, 012, 013 (+ 021, 022 — 규칙 2-1) | w07-014, 015 |
| 불공정영업행위 금지 | w07-016, 017, 018 | w07-019, 020 |
| 부당권유행위 금지 | w07-021, 022, 023 | w07-024, 025, **031** |
| 광고 규제 | w07-026, 027, 028 | w07-029, 030 |

- 위반 문항 18건 + 정상 문항 13건 = **31건**. 라벨 셀은 31 × 6원칙 = **186개**
  (VIOLATION 20 / OK 166 — 정상 라벨이 많은 불균형 구조이므로 정확도만 보지 말고
  **오탐률과 원칙별 재현율을 함께** 봐야 한다).
- 정상 문항 일부는 **오탐 함정**이다. w07-010(예금의 "원금 보장"은 사실),
  w07-019(다른 상품 안내지만 강요 아님), w07-020(정당한 담보 + 3년 이내 수수료는
  허용 예외), w07-025(근거를 밝힌 비교)는 겉보기 신호에 반응하는 모델을 걸러낸다.
- w07-018 ↔ w07-020은 **대조쌍**이다. 중도상환수수료는 3년 경과 후 부과가 위반이고
  3년 이내 부과는 적법하다. 표면적 키워드로는 구분되지 않는다.

## 4. 쟁점 확정 내역 (2026-07-13, 조문 원문 대조 완료)

인덱스에서 조문 원문을 출력해 네 쟁점을 확정했다. 아래는 확정 결론과 근거 조문이다.

### 확정 1 — 단정적 표현: 국면 조건부 복수 라벨

> 제19조③ "…제1항에 따른 설명을 할 때 …사항을 거짓으로 또는 **왜곡(불확실한 사항에
> 대하여 단정적 판단을 제공하거나 확실하다고 오인하게 할 소지가 있는 내용을 알리는
> 행위를 말한다)**하여 설명하거나 …빠뜨려서는 아니 된다."

괄호 정의가 제21조제1호와 문언상 동일함을 확인했다. 다만 적용이 "제1항에 따른 설명을
할 때"로 묶여 있고, 제19조①의 국면은 "권유하는 경우 및 설명을 요청하는 경우"다.

→ **상담(설명 국면)의 단정적 표현은 제21조 + 제19조③ 둘 다 VIOLATION. 광고 카피는
제19조 적용 대상이 아니다.** 판단 기준은 위 **규칙 2-1** 참조.
→ 라벨 변경: w07-021·w07-022의 설명의무 OK → **VIOLATION** (2셀).

### 확정 2 — 광고의 위험 미고지: 제22조만 (현행 유지)

제19조①이 적용 국면을 "권유 / 설명 요청"으로 한정하므로 광고는 포함되지 않는다.
광고의 필수 포함사항 누락·오인 표현은 제22조제3·4항 소관이다.
→ 라벨 변경 없음 (w07-026~028은 광고 규제만 VIOLATION 유지).

### 확정 3 — 부적합 권유는 제17조만, "적합성 회피"가 명시된 경우에만 제21조 복수

부적합 상품 권유 자체는 제17조제3항이다. 제21조로 넘어가려면 **적합성 회피 행위**가
문구에 드러나야 하고, 그 근거 조문은 다음과 같다(모두 제21조제7호를 경유).

| 회피 유형 | 근거 조문 |
|---|---|
| 소비자 정보를 **조작**하여 권유 | 시행령 제16조③제2호 |
| 제17조를 적용받지 않으려고 **"권유를 원하지 않는다"는 의사를 서면으로 받는 행위** (확인서 유도) | 감독규정 제15조⑤제5호 |
| 정보 파악·확인 시 **특정한 답변을 하도록 유도**하는 행위 (성향 유도) | 감독규정 제15조⑤제6호 |

→ **현재 평가셋에 해당 문항 없음.** w07-002는 성향 진단 결과를 *무시*할 뿐, 조작·유도·
확인서 수령 중 어느 것도 문구에 없다. 라벨 변경 없음 (제17조만 VIOLATION).
→ 향후 평가셋 확장 시 위 3유형을 담은 문항을 추가하면 제21조 복수 라벨 대상이 된다.

### 확정 4 — 재권유 금지는 투자성 상품 한정 (w07-022 펀드 유지 + 대조 문항 추가)

> 제21조 "**6. 투자성 상품의 경우** 다음 각 목의 … 나. 계약의 체결권유를 받은
> 금융소비자가 이를 **거부하는 취지의 의사를 표시하였는데도** 계약의 체결권유를 계속하는 행위"

본문이 "투자성 상품의 경우"로 한정됨을 확인했다. 또한 감독규정 제15조④는 시행령
제16조①제3호의 "금융위원회가 정하여 고시하는 기간"을 **1개월**로 정한다(그 기간이 지난
뒤의 재권유는 예외로 허용). w07-022는 "지난주" 재권유이므로 기간 내 → 위반 성립.

→ w07-022는 **펀드(투자성) 유지**.
→ **w07-031 신규 추가**: 예금(비투자성) 거부 후 재권유 대조 문항. 제21조제6호가 적용되지
않고 단정적 표현도 없으므로 **전 원칙 OK**가 기대값이다. 표면 신호("지난주에 안 하신다고
하셨지만 다시 연락")만 보는 모델을 걸러낸다.

## 5. 프롬프트 오염 방지 (평가 신뢰도)

`agent/prompts/judge_v1.0.md`에는 판정 규율을 가르치기 위한 **few-shot 예시**가 원칙별로
1~2개 들어 있다(예: "만기 연장하려면 적금 가입하셔야 합니다" → 불공정영업).

- 이 예시들은 **평가셋에서 가져온 것이 아니라** 조문의 전형(archetype)을 보고 따로 쓴
  것이다. 평가셋 문항은 더 길고 구체적인 시나리오로 독립 작성했다.
- 다만 위반 유형의 원형이 겹치는 것은 불가피하다(6대 원칙의 대표 위반 유형은 정해져
  있다). 따라서 이 평가는 **"가르친 판정 규율을 처음 보는 문구에 적용할 수 있는가"**를
  재는 것이지, zero-shot 성능을 재는 것이 아니다. 수치 해석 시 이 점을 감안해야 한다.
- 평가셋 문항 자체를 프롬프트에 넣는 일은 하지 않았다.
